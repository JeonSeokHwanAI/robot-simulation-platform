# 시뮬레이션 결과 분석 가이드

## 개요
시뮬레이션 실행 결과를 분석하여 보고서를 생성한다.
성능 지표 측정, 문제점 식별, 개선 방향을 도출.

## 분석 항목

### 성능 지표
| 지표 | 설명 | 측정 방법 |
|------|------|-----------|
| 사이클 타임 | 1회 작업 소요 시간 | ROS2 로그 타임스탬프 |
| 성공률 | 작업 성공 횟수 / 전체 | 작업 결과 카운트 |
| 경로 길이 | TCP 이동 총 거리 | 조인트 궤적 적분 |
| 조인트 사용률 | 각 조인트 범위 사용 비율 | 조인트 상태 로그 |

### 안전 지표
| 지표 | 설명 |
|------|------|
| 최소 장애물 거리 | 충돌 여유 |
| 최대 조인트 속도 | 제한 대비 비율 |
| 특이점 근접도 | 야코비안 조건수 |

## 데이터 수집

```bash
# ROS2 bag으로 데이터 기록
ros2 bag record -a -o simulation_data

# 특정 토픽만 기록
ros2 bag record /joint_states /tf -o joint_data
```
