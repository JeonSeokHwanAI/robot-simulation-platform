# 최적화 & 개선안 도출 가이드

## 개요
시뮬레이션 결과 분석(Skill 04)에서 도출된 문제점을 기반으로
성능을 최적화한다.

## 최적화 영역

### 1. 경로 최적화
- 불필요한 경유점 제거
- 직선 경로 → 곡선 경로 (블렌딩)
- 접근/후퇴 방향 최적화

### 2. 속도 프로파일 최적화
- 구간별 속도 조정 (빈 이동 vs 물체 파지 상태)
- 가감속 프로파일 조정
- 조인트 속도 제한 활용률 높이기

### 3. 레이아웃 최적화
- 로봇-작업대 거리 조정
- 소재/제품 위치 최적화
- 도달 범위 활용 극대화

### 4. 작업 시퀀스 최적화
- 작업 순서 변경
- 병렬 동작 (이동 중 그리퍼 동작)
- 대기 시간 최소화

## 최적화 절차

```
1. 현재 성능 측정 (Skill 04 결과)
2. 병목 구간 식별
3. 개선안 생성 (위 영역별)
4. 수정 코드/파라미터 적용
5. 재시뮬레이션
6. 개선 효과 비교
7. 반복 (목표 달성까지)
```

## 결과물
- `{project}/improvements/` 폴더에 개선 이력 저장
- 개선 전후 비교 데이터 포함
