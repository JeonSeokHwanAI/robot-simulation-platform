### ⚠️ 조인트 범위 불일치 — 벤더 확인 필요
두 매뉴얼 간 J3/J6 범위에 대한 기술이 다름:

| 출처 | J3 범위 | J6 범위 |
|------|---------|---------|
| Cobots Manual (p.8) | ±360° | ±165° |
| HW Manual Appendix A (p.53) | ±165° | ±360° |

→ 조인트 속도는 양쪽 동일 (J1,2: 150°/s, J3: 210°/s, J4,5,6: 240°/s)
→ ±165° 제한이 J3인지 J6인지 벤더에게 반드시 확인 필요

## 링크 치수 (✅ 도면 확인됨 — DH-R12-135S)

도면 DH-R12-135S (2024.04.23, V100)에서 추출:

| 구간 | 길이 (mm) | 출처 |
|------|-----------|------|
| 베이스 높이 (바닥→J1) | 164.3 | DH 도면 |
| J1→J2 수평 오프셋 | 176.6 | DH 도면 |
| J2→J3 (상완) | 610.5 | DH 도면 |
| J3→J4 (전완) | 572.5 | DH 도면 |
| 손목 오프셋 | 136 | DH 도면 |
| J5→J6 (손목) | 117 | DH 도면 (2회 표기) |
| J6→플랜지 면 | 106.5 | DH 도면 |

### 도달거리 검증
- 상완 + 전완 = 610.5 + 572.5 = **1,183 mm** → 가변 작업 공간(SR1183)과 일치
- 상완 + 전완 + 손목 = 610.5 + 572.5 + 117 = **1,300 mm** → 최대 도달거리(SR1300)와 일치

## 동역학 파라미터 (🔧 추정값 — 벤더 확인 필요)

| Base | 7.0 | (0, 0, 82.2) | 0.0274 | 0.0274 | 0.0253 |
| Link1 | 6.0 | (0, 0, 70.6) | 0.0229 | 0.0229 | 0.0147 |
| Link2 (상완) | 9.0 | (0, 0, 244.2) | 0.2864 | 0.2864 | 0.0136 |
| Link3 (전완) | 7.0 | (0, 0, 229.0) | 0.1956 | 0.1956 | 0.0088 |
| Link4 (손목1) | 3.2 | (0, 0, 54.4) | 0.0066 | 0.0066 | 0.0032 |
| Link5 (손목2) | 2.3 | (0, 0, 46.8) | 0.0035 | 0.0035 | 0.0018 |

## ROS2 연동 계획
- 공식 ROS2 패키지: 없음
- 외부 제어: TCP/IP SDK 또는 Modbus TCP 경유 가능
- URDF: 3D 파일 + 도면 치수 기반 직접 제작 예정
- MoveIt2: Setup Assistant로 수동 설정 예정
- 가상 모드: ROS2 position_controllers 기반 시뮬레이션
