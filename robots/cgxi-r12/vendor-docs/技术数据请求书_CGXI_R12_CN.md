## 一、请求背景

我们正在基于 **ROS2 (Humble) / Gazebo (Fortress)** 构建 R12-135S 的数字孪生仿真环境，用于：

- 离线轨迹规划与碰撞检测
- 基于 MoveIt2 的运动规划验证
- Pick & Place 等工艺任务的仿真开发

为此，我们需要制作 **URDF（Unified Robot Description Format）** 模型文件。URDF 中的 `<inertial>` 标签要求提供各连杆的**质量、质心坐标和惯性张量**，这些数据直接影响仿真中的动力学精度。

## 四、参考：URDF 数据用途对照

| 请求项目 | URDF 标签 | 用途 |
|----------|----------|------|
| 关节运动范围 | `<limit lower="..." upper="..."/>` | 运动规划边界约束 |
| 连杆质量 | `<mass value="..."/>` | 动力学仿真（重力/惯性） |
| 质心坐标 | `<origin xyz="..." rpy="..."/>` | 动力学仿真（力矩计算） |
| 惯性张量 | `<inertia ixx="..." iyy="..." .../>` | 动力学仿真（角加速度） |
| DH 参数 | `<joint>` + `<link>` 的几何关系 | 正/逆运动学 |
