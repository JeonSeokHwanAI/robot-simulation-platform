## 데이터 추출 현황

### ✅ 확인 완료
- 기본 사양 (하중, 도달거리, 정밀도, 속도, 자중, 전력 등)
- 조인트 속도 (J1,2: 150°/s, J3: 210°/s, J4,5,6: 240°/s)
- 링크 치수 (DH 도면: 164.3, 610.5, 572.5, 176.6, 136, 117, 106.5 mm)
- 작업 공간 (최대 SR1300, 가변 SR1183)
- 베이스/플랜지 기계 치수 및 볼트 패턴
- 통신 인터페이스 (EtherCAT, Modbus TCP/RTU, RS232, RS485, TCP/IP SDK)
- I/O 사양 (디지털/아날로그/안전/인코더, 캐비닛 및 툴 사이드)
- 안전 기능 (비상정지, 안전정지, 충돌감지, 브레이크 토크)
- 제어 캐비닛 K20 사양

### ⚠️ 불일치 발견
- **J3/J6 동작 범위**: Cobots Manual은 J6=±165°, HW Manual은 J3=±165°

### ❌ 미확인 (벤더 요청 필요)
- 정확한 DH 파라미터 테이블 (a, d, α, θ)
- 각 링크 질량, 무게중심, 관성 모멘트
- TCP/IP SDK 상세 문서 (Advanced User Guide)
- 제어 캐비닛 기본 IP 주소
