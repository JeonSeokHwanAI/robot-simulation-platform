## ROS2 연동

- 공식 ROS2 패키지: ✅ doosan-robot2
- URDF: 공식 패키지 내 포함
- MoveIt2: 공식 패키지 내 포함 (OMPL/CHOMP/Pilz 플래너)
- 시뮬레이션: DRCF 에뮬레이터 (Docker)

## DRCF 에뮬레이터

두산 로봇 제어기 에뮬레이터(DRCF)를 Docker 컨테이너로 실행하여
실제 로봇 없이 시뮬레이션 가능.
